# LaundryKu

LaundryKu adalah aplikasi Android sederhana untuk asesmen mobile programming. Aplikasi ini dibuat dengan Kotlin dan Jetpack Compose, menggunakan minimum SDK 26, serta menampilkan alur input data pelanggan, pemilihan layanan laundry, dan ringkasan total harga.

## Fitur

- Top App Bar
- Gambar banner laundry
- Input nama pelanggan
- Input berat laundry
- Pilihan layanan dengan radio button
- Validasi input sederhana
- Perhitungan total harga
- Halaman tentang aplikasi
- Bagikan hasil ke aplikasi lain

## Struktur Screen

1. Main Screen
2. About Screen

## Cara Membuka

1. Buka Android Studio.
2. Pilih **Open**.
3. Arahkan ke folder project ini.
4. Tunggu proses Gradle Sync.

## Catatan

- Jika Android Studio meminta membuat atau memperbarui Gradle Wrapper, izinkan saja.
- Jika SDK belum cocok, gunakan compile SDK 34 dan minimum SDK 26.
